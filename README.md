# Getting Started
To run JavaPaint, you need to clone this Github repository or download it as a zip file to your local machine. Compile and run <tt>JavaPaint.java</tt> with Java 8.0 or higher to launch the program.

## Dependencies Needed:
- Java 8.0 or higher
